# CHServiceTime

This is a refactor of ServiceTime to support android 5.0+.

The application demostrates a couple of technologies:

- MVP achitecture
- RxJava/RxAndroid
- SQLBrite and SQLDelight
- Retrofit and Gson
- AutoValue
- Firebase Authentication``
